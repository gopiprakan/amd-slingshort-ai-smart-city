# Smart City AI - Government Dashboard

A modern government dashboard application for smart city management, built with React and Vite.

## Features

- Interactive city map with Leaflet integration
- Real-time data visualization with Recharts
- Responsive design with Bootstrap
- Heat map visualization for city analytics
- Modern UI with government-themed styling

## Tech Stack

- **React** 19.2.0
- **Vite** 7.3.1
- **Leaflet** - Interactive maps
- **Recharts** - Data visualization
- **Bootstrap** 5.3.8 - UI framework

## Installation

1. Clone the repository:
```bash
git clone <your-repository-url>
cd government-dashboard
```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```

4. Open your browser and navigate to `http://localhost:5173`

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint
